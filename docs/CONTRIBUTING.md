# Contributing to E-Skimming Labs

Thank you for your interest in contributing! This project is educational and
aims to help security professionals understand e-skimming attacks.

## How to Contribute

1. **New Labs**: Propose new attack scenarios
2. **Detection Rules**: Add Semgrep/YARA rules
3. **Documentation**: Improve explanations
4. **Bug Fixes**: Fix issues in existing labs

## Guidelines

- All code must be for educational purposes only
- Include detailed documentation
- Test thoroughly before submitting
- Follow existing code style
- Add appropriate warnings about malicious code

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test your changes
5. Submit a pull request with clear description

## Code of Conduct

- Focus on education and defense
- No discussion of real-world attacks on live systems
- Professional and respectful communication
- Report security issues privately
